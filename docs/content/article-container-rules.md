# Правила для контейнера `article-content`

Каждая статья-категория строится вокруг корневого контейнера:

```tsx
const PsFromNewbies: React.FC = () => {
  return <div className="article-content">{/* спойлеры */}</div>;
};
```

Класс `article-content` отвечает за:

- отступы и типографику абзацев;
- оформление списков, `Addition`, `ArticleMedia` и вложенных спойлеров;
- корректную работу затемнения при открытом `DetailsSummary`.

> Не заменяйте этот контейнер собственными обёртками и не убирайте класс: на нём завязаны стили и поведение.
