# Правила для `HostsAdobe`

Файл компонента: `src/components/content/HostsAdobe.tsx`.

## Правила использования

- вставляйте блок только в сценариях, где действительно нужна правка `hosts`;
- перед блоком объясняйте последствия и цель изменения;
- не копируйте список доменов вручную, используйте компонент целиком.

## Чеклист

- блок размещён внутри `DetailsSummary`;
- рядом есть пояснение, зачем пользователь выполняет эти действия;
- в ответе есть предупреждение о рисках при редактировании `hosts`;
- проверено, что при недоступности внешнего источника текст не вводит в заблуждение.

## Антипаттерны

- предлагать правку `hosts` как универсальное решение для любых проблем;
- вставлять список доменов вручную вместо компонента;
- использовать блок без контекста и без предупреждения о последствиях.
