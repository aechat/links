# Правила для feature-компонентов

Компоненты: `SearchEngine`, `EasingEditor`, `ConverterJsonToTgs`, `ConverterTgsToJson`.

## Правила использования

- `SearchEngine` подключайте там, где есть индексируемый контент и поиск по странице;
- `EasingEditor` используйте в интерактивных сценариях настройки кривых;
- конвертеры используйте только для целевых форматов Lottie/TGS.

## Чеклист

- компонент решает прикладную задачу, а не дублирует контентный блок;
- пользовательский сценарий для компонента описан в тексте страницы;
- для конвертеров проверена направленность преобразования (`JsonToTgs` или `TgsToJson`).

## Антипаттерны

- подключать feature-компонент «на всякий случай» без сценария использования;
- смешивать в одном UI независимые инструменты без явной связи;
- использовать конвертер не по назначению формата.
