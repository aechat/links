# Правила для `Divider`

`Divider` помогает разбивать длинный ответ на логические блоки.

## Когда использовать

- в одном `DetailsSummary` есть несколько самостоятельных подпунктов;
- нужно визуально отделить этапы решения.

## Правила

- текст в `Divider` короткий и предметный;
- не ставьте `Divider` между каждым абзацем;
- если `Divider` слишком много, возможно, ответ стоит разбить на отдельные спойлеры.

## Пример

```tsx
import {Divider} from "antd";

...
<Divider>Проверяем настройки экспорта</Divider>;
...
```
