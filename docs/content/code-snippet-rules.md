# Правила для `CodeSnippet`

Файл компонента: `src/components/content/CodeSnippet.tsx`.

## Когда использовать

- команды терминала;
- фрагменты кода и выражения;
- конфиги и блоки, которые пользователь должен быстро скопировать.

## Как работает компонент

- `children` приводится к строке (включая вложенные узлы и `<br />`).
- Подсветка выполняется через `highlight.js`.
- Клик по блоку копирует весь текст в буфер обмена.
- `language` по умолчанию: `javascript`.

## Практика

- Не используйте `CodeSnippet` для коротких inline-фрагментов в 1–2 слова, для этого лучше использовать `<mark className="copy">...</mark>`.
- Указывайте `language`, если это не JavaScript.
- Давайте рабочие команды без лишнего шума, при необходимости добавляйте комментарии.

## Чеклист

- блок действительно требует копирования, а не inline-выделения;
- `language` задан, если это не JavaScript;
- команда или код проверены и соответствуют реальному сценарию.

## Антипаттерны

- помещать в `CodeSnippet` короткие фразы без пользы для копирования;
- добавлять в блок нерабочие или непроверенные команды;
- смешивать в одном snippet несколько несвязанных сценариев.

## Пример

```tsx
import CodeSnippet from "../../../components/content/CodeSnippet";

<CodeSnippet language="bash">{`yarn install
yarn dev`}</CodeSnippet>;
```
