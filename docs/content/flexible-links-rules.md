# Правила для `.flexible-links`

Стили: `src/styles/components/_flexible-links.scss`.

## Когда использовать

- подборка однотипных ссылок по теме;
- компактный список ресурсов в формате кнопок.

## Разметка

Используйте контейнер с классом `flexible-links`, внутри размещайте ссылки `a` или кнопки `button`.

```tsx
<div className="flexible-links">
  <a href="https://example.com/course">Курс для новичков</a>
  <a href="https://example.com/channel">Полезный канал</a>
</div>
```

## Поведение

- элементы отображаются как единая группа кнопок;
- в `DetailsSummary` поддерживается long-press по ссылке для копирования URL;
- `SearchEngine` индексирует ссылки из `.flexible-links a` как отдельный блок.

## Практика

- подписи ссылок должны объяснять, что именно откроется;
- не используйте «сюда», «тут», «линк»;
- длинные URL не вставляйте как текст, оставляйте их в `href`.
