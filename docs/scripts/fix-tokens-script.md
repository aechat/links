# Скрипт `fixTokens`

Файл: `scripts/fixTokens.js`

## Назначение

Интерактивно исправляет токены в `.tsx`-файлах:

- приводит содержимое тегов `<file>`, `<image>`, `<video>`, `<audio>` к верхнему регистру;
- нормализует пробелы вокруг `+` в `<key>`.

## Запуск

```bash
node scripts/fixTokens.js
```

## Режим работы

1. Скрипт предлагает выбрать категории обработки (`file/image/video/audio/key`).
2. Для каждого изменения спрашивает подтверждение:
   - `y` — заменить только это вхождение;
   - `n` — пропустить это вхождение;
   - `a` — автоматически подтверждать такую замену в текущей сессии;
   - `s` — добавить исходное слово в whitelist (пропускать все его вхождения).

## Технические детали

- Использует общий раннер [`script-runner-script.md`](./script-runner-script.md).
- Обрабатывает только `.tsx`.
- Все действия логируются в `scripts/script_log.txt`.
