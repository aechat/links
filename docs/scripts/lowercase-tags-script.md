# Скрипт `lowercaseTags`

Файл: `scripts/lowercaseTags.js`

## Назначение

Интерактивно приводит значения атрибута `tag="..."` к нижнему регистру в `.tsx`-файлах.

## Запуск

```bash
node scripts/lowercaseTags.js
```

## Как работает

1. Ищет в файле все вхождения `tag="..."`.
2. Вычисляет предложенный вариант в нижнем регистре.
3. Для каждого изменения спрашивает подтверждение через `askForConfirmation`.
4. Записывает файл только при подтверждённых изменениях.

## Технические детали

- Использует общий раннер [`script-runner-script.md`](./script-runner-script.md).
- Работает только для `.tsx`.
- Логи пишутся в `scripts/script_log.txt`.
