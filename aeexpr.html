Для чего нужны выражения и в каких случаях их используют?

Выражения, экспрешены или expressions — полезная функция в Adobe After Effects, с помощью которой можно задать анимацию или трансформацию свойств через JavaScript или ExtendScript (устаревший). Проще говоря, это короткий фрагмент кода, «прикреплённый» к конкретному свойству слоя: он вычисляется на каждом кадре и подставляет нужное значение вместо множества ключей. В отличие от скриптов формата JSX и JSXBIN, которые управляют всем проектом, выражения работают именно с выбранным свойством. Например, если вы хотите, чтобы яркость изображения плавно мерцала, достаточно прописать выражение wiggle(ЗначениеЧастоты, ЗначениеАмплитуды)— и Adobe After Effects сам будет менять значение яркости по заданной формуле.

Отчасти синтаксис JavaScript, используемый в Adobe After Effects, похож на синтаксис, который вы могли видеть в браузерных скриптах. Однако есть различия в обращении к свойствам, циклах и поддержке некоторых функций. Очевидно, что console.log("Hello, aeexprfaq!") в выражениях работать не будет.

Как включить режим написания выражений?

Чтобы включить режим работы и написания выражения для свойства слоя в Adobe After Effects, достаточно кликнуть по иконке секундомера с зажатой клавишей Alt. После этого откроется поле для ввода кода выражения.

Вы также можете включить режим отображения графика скорости или значений, который отображает текущее значение или скорость вашего выражения, с помощью соответствующей кнопки.

Для чего нужны комментарии в коде выражений? Код внутри комментариев может выполняться?

Комментарии - необязательный участок кода, в которую можно записать различную полезную информацию для себя или для тех, кто планирует воспользоваться вашим выражением. Всё, что вы напишите внутри комментариев - будет игнорироваться и не будет выполняться.

if (time > 3) {
  // ставим условие, что если текущее время больше 3 секунд, то вывести текст
  ("Прошло больше, чем 3 секунды");
} else if (time == 3) {
  // если текущее время ровно 3 секунды, то вывести число
  (3);
} else {
  // иначе, если предыдущие условия не выполнены, то вывести другой текст
  ("Прошло меньше чем 3 секунды");
}

Как применить действие выражения только для одной оси?

wiggle по X Y Zзадать значение для одного элемента массива

Бывают случаи, когда вам нужно применить значение какой-либо переменной или функции только к одному элементу в массиве, например в положении или якорной точке. Обычно у положения, если их не делить через Separate Dimensions, есть два или три элемента в массиве. Они представлены как оси X и Y, если слой в двумерном пространстве, или оси X, Y и Z, если слой - в трёхмерном.

Например, если вам нужно применить wiggle() только для оси X, Y или Z, то вам достаточно написать одно из простых выражений в последнюю строку редактора, где вместо значений freq и amp вы можете вставить свои или заменить wiggle() на свою переменную. Значение value[0],value[1] или value[2] возвращают то значение, которое вы записываете в атрибуте положения.

// функция wiggle() будет применяться только для оси X, значение оси Y не изменяется
[wiggle(freq, amp)[0], value[1]]
// функция wiggle() будет применяться только для оси Y, значение оси X не изменяется
[value[0], wiggle(freq, amp)[1]]
// функция wiggle() будет применяться только для оси Z, значения осей X и Y не изменяется
[value[0], value[1], wiggle(freq, amp)[2]]

Если у атрибута можно записать только одно значение, то достаточно воспользоваться value к тому значению, которое вам нужно. value возвращает число, поэтому с ним можно выполнять различные математические операции, по типу умножения или вычитания.

Как я могу отформатировать текст в строке?

С помощью форматирования строки вы можете преобразовать текст как обычную строку, но с подставленными значениями переменных. Это можно сделать двумя способами.

  • Через $ и обратную кавычку -
  • Через конкатенацию - 
/* !! ДАННОЕ ВЫРАЖЕНИЕ ПРИМЕНЯЕТСЯ К ТЕКСТОВОМУ СЛОЮ В SOURCE TEXT !!
Данный код возвращает текущее время в композиции в формате "секунда и кадр" путём форматирования текста двумя способами.
В конце выводится переменная строкового типа, отформатированная через знак доллара и обратные "кавычки".
*/
totalFrames = timeToFrames(time); // записываем значение всего количества кадров
fps = timeToFrames(1); // записываем значение количества кадров за одну секунду
seconds = Math.floor(time); // округляем значение времени до целых секунд
frames = totalFrames % fps; // вычисляем остаток количества кадров
timePrefix = "Текущее время: "; // префикс для текста
function plural(n, forms) { // необязательная функция для склонения текста в зависимости от числа
  if (n % 10 == 1 && n % 100 != 11) return forms[0];
  else if (n % 10 >= 2 && n % 10 <= 4 && (n % 100 < 10 || n % 100 >= 20)) return forms[1];
  else return forms[2];
}
// форматирование через знак доллара и обратные "кавычки"
inlineFormattedText = `${timePrefix}${seconds} ${plural(seconds, ["секунда", "секунды", "секунд",])} и ${frames} ${plural(frames, ["кадр", "кадра", "кадров"])}`;
if (seconds == 0 && frames == 0) {
  inlineFormattedText = "Начало композиции";
} else if (frames === 0) {
  inlineFormattedText = `${timePrefix}${seconds} ${plural(seconds, ["секунда", "секунды", "секунд"])}`;
}
// форматирование через конкатенацию
concatFormattedText = timePrefix + seconds + " " + plural(seconds, ["секунда", "секунды", "секунд"]) + " и " + frames + " " + plural(frames, ["кадр", "кадра", "кадров"]);
if (seconds == 0 && frames == 0) {
  concatFormattedText = "Начало композиции";
} else if (frames === 0) {
  concatFormattedText = timePrefix + seconds + " " + plural(seconds, ["секунда", "секунды", "секунд"]);
}
// возвращаем нужное значение
inlineFormattedText; // или concatFormattedText
